Política de privacidad de DGT Monitor

Última actualización: 15 de mayo de 2026

DGT Monitor es una aplicación no oficial para consultar incidencias de tráfico publicadas por la Dirección General de Tráfico y otros servicios relacionados con mapas y rutas. La aplicación no representa a la DGT ni a ningún organismo público.

Esta política explica qué datos se tratan al usar DGT Monitor.

Responsable y contacto

Responsable de la aplicación: Daniel Pazos

Contacto: danielpazos.dev@gmail.com

Datos que guarda la aplicación en el dispositivo

DGT Monitor puede guardar localmente en el dispositivo:

  • carreteras seleccionadas;
  • carreteras favoritas;
  • zonas de vigilancia creadas por el usuario;
  • rutas o viajes guardados;
  • preferencias de ajustes, tema y notificaciones;
  • histórico reciente de incidencias mostradas.

Estos datos se usan para que la aplicación recuerde la configuración del usuario y funcionan principalmente dentro del propio dispositivo.

Datos de incidencias y mapas

Para mostrar incidencias, mapas y rutas, la aplicación realiza conexiones a servicios externos, entre ellos:

  • Dirección General de Tráfico, para consultar incidencias;
  • OpenStreetMap, Overpass, Nominatim y OSRM, para mapas, carreteras, búsqueda de lugares y cálculo de rutas;
  • Google Maps, solo cuando el usuario pulsa la opción de abrir una incidencia en mapas.

Cuando se usan funciones de ruta o mapas, los servicios externos pueden recibir los datos técnicos necesarios para responder a la petición, como direcciones o lugares buscados, coordenadas de una incidencia o información de red del dispositivo.

Firebase Analytics

DGT Monitor usa Firebase Analytics para entender el uso general de la aplicación y mejorarla. Se pueden registrar eventos como búsquedas, pantallas abiertas, funciones utilizadas, apertura de detalles, uso del mapa o cambios de ajustes.

Los eventos de Analytics no incluyen deliberadamente nombres, correos electrónicos, rutas exactas ni coordenadas personales. Google Firebase puede tratar identificadores del dispositivo o de la instalación, datos técnicos y métricas de uso de acuerdo con sus propias condiciones y documentación.

Firebase Crashlytics

DGT Monitor usa Firebase Crashlytics para detectar fallos y mejorar la estabilidad de la aplicación.

Cuando ocurre un fallo, Crashlytics puede recoger información técnica como el modelo del dispositivo, sistema operativo, versión de la aplicación, estado del fallo, trazas de error, identificadores de instalación y datos de diagnóstico relacionados.

Notificaciones

La aplicación puede enviar notificaciones locales para avisar de incidencias nuevas si el usuario activa esta función. Las notificaciones se generan desde el dispositivo a partir de las incidencias consultadas.

Compartición de datos

DGT Monitor no vende datos personales.

Los datos pueden ser tratados por proveedores necesarios para el funcionamiento de la aplicación, como Google Firebase, servicios de mapas y rutas, o servicios externos de consulta de incidencias.

Conservación

Los datos guardados localmente permanecen en el dispositivo hasta que el usuario los borra, cambia la configuración, desinstala la aplicación o el sistema elimina sus datos.

Los datos tratados por servicios externos se conservan según las políticas de dichos proveedores.

Derechos y solicitudes

El usuario puede borrar los datos locales de la aplicación desde los ajustes del sistema Android, desinstalando la aplicación o usando las opciones internas disponibles, como borrar el histórico.

Para cualquier solicitud relacionada con privacidad, puede contactar en: danielpazos.dev@gmail.com

Cambios en esta política

Esta política puede actualizarse para reflejar cambios en la aplicación, en sus servicios o en requisitos legales. La versión vigente será la publicada junto a la aplicación.
